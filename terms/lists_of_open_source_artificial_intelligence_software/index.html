Nyílt forráskódú mesterséges intelligencia szoftverek listái

term_id: lists_of_open_source_artificial_intelligence_software

Category: basic_concepts

Definition

Ezek szervezett adattárakra utalnak, mint például GitHub témákra, ‘Awesome’ listákra vagy közösség által karbantartott wikikre, amelyek összegyűjtik a mesterséges intelligenciával kapcsolatos nyílt forráskódú szoftvereket. Ezek alapvető erőforrásokként szolgálnak.

Summary

Kurátorként gondozott gyűjtemények vagy könyvtárak, amelyek nyilvánosan elérhető AI eszközöket, könyvtárakat és keretrendszereket sorolnak fel.

Key Concepts

  • Nyílt forráskód
  • Erőforrás-kurátorás
  • Közösségi karbantartás
  • Szoftverfelfedezés

Use Cases

  • Kutatási eszközök felfedezése
  • Indítvállalatok technológiai stackjének kiválasztása
  • Oktatási erőforrások gyűjtése
