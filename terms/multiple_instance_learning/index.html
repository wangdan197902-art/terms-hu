Több példányos tanulás

term_id: multiple_instance_learning

Category: training_techniques

Definition

A több példányos tanulás (MIL) olyan helyzeteket kezel, ahol az adatokat ’táskákba’ csoportosítanak egyetlen címkével, miközben a táskákon belüli egyedi példányok címkézetlenek maradnak. Egy táska általában pozitív, ha legalább egy benne lévő példány pozitív, ellenkező esetben negatív.

Summary

Egy gyengén felügyelt tanulási paradigmában a címkéket nem az egyedi példányokra, hanem a példányok csoportjaira (táskákra) rendelik.

Key Concepts

  • Taskszintű címkézés
  • Példányszintű bizonytalanság
  • Gyenge felügyelet
  • Pozitív/negatív táska logika

Use Cases

  • Gyógyszerhatásosság előrejelzése
  • Képosztályozás határmezőkkel
  • Tartalomalapú képvisszakeresés
