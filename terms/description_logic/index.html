Leíró logika

term_id: description_logic

Category: engineering_practice

Definition

A leíró logikák (DL) az elsőrendű logika döntőképes részhalmazai, amelyek az ontológiák, különösen a Web Ontológia Nyelv (OWL) elméleti alapját képezik. Lehetővé teszik a fogalmak precíz definiálását és hierarchikus szervezését, valamint a logikai következtetések automatizált elvégzését az ontológiában tárolt tudás alapján.

Summary

Formális tudásreprezentációs nyelvek családa, amelyeket a domének fogalmi szerkezetének leírására és annak alapján történő következtetésre használnak.

Key Concepts

  • Ontológia
  • Következtetés
  • Szubsumpció (Alárendeltség)
  • Fogalmi hierarchia

Use Cases

  • Szemantikus web ontológiák létrehozása
  • Orvosi tudásgráfok
  • Automatikus osztályozó rendszerek

Related Terms

  • OWL (Web Ontology Language - Web Ontológia Nyelv)
  • RDF (Resource Description Framework - Erőforrás-leíró keretrendszer)
  • Tudásgráf (Knowledge Graph)
  • Szemantikus web (Semantic Web)
