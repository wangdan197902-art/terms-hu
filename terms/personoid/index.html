Personoid

term_id: personoid

Category: basic_concepts

Definition

Egy personoid egy entitás, legyen szó robotikáról vagy virtuálisról, amelyet úgy mérnökölték, hogy hasonlítson vagy viselkedjen emberként. A robotikában ez fizikai antropomorfizmust jelent, míg az AI-ban gyakran chatbotokra vagy virtuális

Summary

Egy humanoid robot vagy szoftverügynök, amelyet úgy terveztek, hogy utánozza az emberi fizikai megjelenést vagy beszélgetési stílust a természetes interakció elősegítése érdekében.

Key Concepts

  • Antropomorfizmus
  • Humanoid robotika
  • Virtuális ügynökök
  • Rettenetes völgy (Uncanny Valley)

Use Cases

  • Ügyfélszolgálati recepciósok
  • Idősgondozó társak
  • Oktatói tutorok
