Adathalmaz: Code Search Net

term_id: datasetcode_search_net

Category: application_paradigms

Definition

A Code Search Net egy átfogó adathalmaz, amelyet a kódvisszaállítás kutatásának előmozdítására hoztak létre. Több millió természetes nyelvű leírás és a hozzájuk tartozó Java kódrészletek párját tartalmazza. Ez az adatbázis…

Summary

Egy benchmark adathalmaz, amelyet arra terveztek, hogy értékelje a kódkereső rendszereket természetes nyelvű lekérdezések és a hozzájuk tartozó releváns kódrészletek párosításával.

Key Concepts

  • Kódvisszaállítás
  • Szemantikus keresés
  • Java kód
  • Természetes nyelvfeldolgozás

Use Cases

  • IDE bővítmények fejlesztése
  • Kódértelmezési kutatások
  • Automatikus kód dokumentáció
