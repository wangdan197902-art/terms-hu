Discovery System

term_id: discovery_system

Category: basic_concepts

Definition

A felfedezési rendszer egy számítási keretrendszer, amelynek célja a tudományos vagy elemzési áttörések felgyorsítása az óriási adatterek automatikus feltárásával. Ellentétben a hagyományos optimalizációval, amely csak a meglévő paraméterek finomhangolására fókuszál…

Summary

Egy automatizált keretrendszer vagy algoritmikus folyamat, amelyet arra terveztek, hogy azonosítsa, feltárja és kiértékelje a lehetséges megoldásokat, mintázatokat vagy hipotéziseket egy nagy keresési térben.

Key Concepts

  • Automatikus feltárás
  • Hipotézisgenerálás
  • Aktív tanulás
  • Keresési tér navigáció

Use Cases

  • Új molekuláris szerkezetek azonosítása gyógyszerek fejlesztéséhez
  • Új anyagok felfedezése specifikus tulajdonságokkal
  • Váratlan korrelációk megtalálása nagy adathalmazokban

Related Terms

  • Automated Machine Learning (Automatizált gépi tanulás / AutoML)
  • Scientific Discovery (Tudományos felfedezés)
  • Bayesian Optimization (Bayes-optimalizálás)
  • Data Mining (Adatbányászat)
